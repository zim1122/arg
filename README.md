Method 1: Figure Window থেকেই change করা
আগে তোমার graph plot(t, m) দিয়ে run করো
Graph window open হবে
এখন:
graph line এর উপর double click করো
অথবা right click → Properties
“Property Inspector” open হবে
এখানে খুঁজো:
Marker → সেট করো o (circle)
Marker Size → ইচ্ছামতো বাড়াও (e.g., 6–8)
চাইলে Marker Face Color → color fill করতে পারো

👉 তাহলে graph এ dot দেখা যাবে (sampling points)
